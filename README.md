# AI Personalized Marketing Video Ad Agent

A Streamlit app that generates personalized marketing assets for shoe products using a 3-step GenAI pipeline:

1. Generate a personalized slogan and product description  
2. Generate a cinematic video script  
3. Generate a short ad video from the product image and cinematic video script

## Features

- Customer profile input (name, age, gender, nationality, location)
- Product selection with image preview
- AI-generated slogan and description
- AI-generated cinematic script
- AI-generated video output
- Simple Streamlit interface
